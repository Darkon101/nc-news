NC NEWS API

API Description

This is a RESTful API. This API accepts JSON-encoded request bodies, returns JSON-encoded responses, and uses standard HTTP response codes and REST verbs.

These endpoints constitute core functionalities of the application.

Connecting to the API

API Root Path: https://nc-news-ldgq.onrender.com/api
